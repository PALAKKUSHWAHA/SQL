Task:

Write a query to:

Combine first_name, last_name, and dept_name into one column as a single string.

Use JOIN to connect both tables.

Use GROUP BY to show count of employees in each department.

âœ… Solution Query
SELECT 
    d.dept_name,
    CONCAT(e.first_name, ' ', e.last_name, ' - ', d.dept_name) AS employee_info,
    COUNT(e.emp_id) AS total_employees
FROM Employees e
JOIN Departments d 
    ON e.dept_id = d.dept_id
GROUP BY d.dept_name, e.first_name, e.last_name;

Write an SQL query to:

Combine first_name, last_name, and city_name into one column named customer_info.

Display the count of customers in each city.

Use JOIN, string functions, and GROUP BY.

âœ… Solution
SELECT 
    c.city_name,
    CONCAT(cu.first_name, ' ', cu.last_name, ' - ', c.city_name) AS customer_info,
    COUNT(cu.cust_id) AS total_customers
FROM Customers cu
JOIN Cities c 
    ON cu.city_id = c.city_id
GROUP BY c.city_name, cu.first_name, cu.last_name;
